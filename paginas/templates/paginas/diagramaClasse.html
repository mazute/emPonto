Diagrama de Classe

EmPonto

O diagrama de classes é uma representação da estrutura e relações das classes que servem de modelo para objetos

As classes recebem os atributos e métodos do sistema, os atributos definem as características do sistema e possuem identificação e tipo do dado, os métodos ela definem as ações e comportamentos do sistema e possuem nome e parâmentros.

O relacionamento entre as classes definem como elas se interagem uma com as outras no sistema.

Usumos o padrão MVC para separar representações de informação internas dos modos como a informação é apresentada para e aceita pelo usuário, levando ao desenvolvimento paralelo de maneira eficiente.
